Runs in browser and can be installed as a tablet app (PWA).

Download / install on a tablet

This is a Progressive Web App: there is no app-store package to download. Open the hosted page in your tablet browser, then install it to the home screen.

  1. Host this folder over HTTPS, or use http://localhost while developing.
  2. Open the app URL in Chrome, Edge, or Safari on the tablet.
  3. Use the browser install action: Chrome/Edge show Install app; Safari uses Share → Add to Home Screen.

If the install button is hidden, use your browser menu or Share sheet to add the app to the home screen.

Connection

If you pick proxy, set your server env variable OPENAI_API_KEY and leave the key field blank here.
