Install on a tablet

This Progressive Web App installs from the browser; no app-store package is required.

  1. Host this folder over HTTPS, or use http://localhost while developing.
  2. Open the app URL in Chrome, Edge, or Safari on the tablet.
  3. Use Install app or Share → Add to Home Screen.

If the install button is hidden, use your browser menu or Share sheet.
